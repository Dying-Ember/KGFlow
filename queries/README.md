# Tree-sitter Query Files

Organized by language and concept domain:

| Domain | File | Purpose |
|--------|------|---------|
| Definitions | `defs.scm` | Function, class, method declarations |
| Imports | `imports.scm` | Import/module statements |
| Calls | `calls.scm` | Function and method call expressions |
| Control Flow | `control_flow.scm` | if/try/for/while, return/raise/throw/defer/go |
